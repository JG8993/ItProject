Program Requirements

The courses listed below are a suggested sequence; as long as prerequisites are met, courses may be taken in a different order.

Quarter One

  • CSD 110 - Computer Programming Fundamentals with Python
  • CSD 112 - HTML and CSS
  • ENGL& 101 - English Composition I

Quarter Two

  • CS& 141 - Computer Science I Java
  • CSD 122 - JavaScript
  • CSD 138 - Structured Query Language (SQL)

Quarter Three

  • CS 143 - Computer Science II Java
  • CSD 268 - Quality Assurance Methodologies
  • MATH& 141 - Pre-Calculus I

Quarter Four

  • ART 102 Introduction to 2D Design
  • PHYS& 114 General Physics I w/Lab
  • SOC& 101 Introduction to Sociology

Quarter Five

  • CSD 230 - Programming for Mobile Devices
  • PHP Scripting
  • C++ Programming

Quarter Six

  • CSD 228 Programming with C#
  • CSD 298 Technical Interview/Job Seach
  • OR
  • DSGN 290 Portfolio/Job Search
  • CSD 297 IT Project
  • Technical Elective CSD 235 or CSD 294

General Education Requirements

  • ENGL& 101 - English Composition
  • MATH& 141 - Pre-Calculus (or any higher-level course in the calculus sequence)
  • PHYS& 114 - General Physics I w/Lab (or any Natural Science course with a lab)
  • SOC& 101 - Introduction to Sociology (or any Social Science course)
  • ART 102 - Introduction to 2D Design (or any Humanities course)
